# Dian Azura — Post Generator

Generate Instagram captions in Dian Azura's voice. Powered by Claude Sonnet + 1,656 real Instagram captions.

## Live app
👉 `https://YOUR-USERNAME.github.io/da-generator`

## Features
- Short / Medium / Long caption tiers
- Hashtag sets for each tier
- Photo upload (camera or library)
- Emotional tone + post type selector
- API key saved locally on device

## Setup
1. Fork or push this repo to GitHub
2. Go to Settings → Pages → Source: GitHub Actions
3. Open the live URL in Safari on iPhone
4. Add to Home Screen for native app feel
5. Paste your Anthropic API key (sk-ant-...) once — it saves on device
